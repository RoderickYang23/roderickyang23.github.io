1. Motivation & Research

The Why: The Boredom of Collaboration

College group projects are notorious for "划水" (slacking) or "一个人carry" (one-person show). Traditional tools like Notion are too serious. Our group chose the **Social Track** to address this social pain point using a playful, metaphorical design from RPGs, turning embarrassing task delegation into exciting adventures.

The Gap

Instead of summarizing 4 papers and 4 products, we list what they did well and what they missed:

  • Pros: Clear tracking, mobile access, feature-rich.
  • Cons (THE GAP): Boring UI, lack of immediate emotional feedback, high learning curve for metaphors.

The Stakeholders

Primary: Students. Secondary: Teachers (who could see the "伤害排行榜" - contribution ranking).

Requirements for "Playful"

We compared the Tavern metaphor with a Space Exploration metaphor. We chose the tavern because it's more localized, cozy (Stardew-vibe), and easier to implement using traditional UI metaphors (buttons = scrolls).

4. Technical Implementation

System Architecture

[美化点：上传架构图 images/architecture.png]

Since this is just for demonstration, we will not build a full backend. All task data will be stored as a JSON array in the browser's `localStorage`. This makes the implementation very easy (React.js + pure CSS).

Recommended Libraries

We plan to use `dnd-kit` for silky smooth drag-and-drop animation, and `canvas-confetti` to trigger full-screen celebrations when the boss is defeated.
